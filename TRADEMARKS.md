The Filament name, logo, and related brand assets are not licensed under the open-source license for this repository.

## What Is Allowed

You may use the project name in a purely descriptive way, such as:

- saying that your project uses Filament
- linking to this repository
- writing factual comparisons or reviews

## What Requires Permission

You may not, without prior written permission:

- imply endorsement by the Filament project
- use Filament branding for a fork in a confusing way
- use project logos in commercial marketing materials as if they were yours
- register domains, package names, or social handles likely to confuse users about project ownership

## Reason

The code is open source.
The brand remains controlled so the project can protect reputation, distribution quality, and future commercial positioning.
