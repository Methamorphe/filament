# Code of Conduct

## Our Standard

Filament is committed to a respectful, technically serious environment.

Participants in project spaces are expected to:

- act professionally
- argue in good faith
- focus criticism on ideas, code, and tradeoffs
- avoid harassment, intimidation, or personal attacks
- respect differing levels of experience and background

## Unacceptable Behavior

This includes:

- personal insults
- discriminatory language
- deliberate harassment
- doxxing
- sustained bad-faith disruption
- sexualized language or attention in project spaces

## Scope

This applies in:

- issues
- pull requests
- discussions
- project chat spaces
- public events or social posts directly representing the project

## Enforcement

Project maintainers may remove comments, close threads, reject participation, or ban users whose behavior violates this policy.

Reports can be made privately to the repository owner via GitHub.

## Project Ethos

Strong technical disagreement is acceptable.
Contempt is not.
